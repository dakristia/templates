## Prerequisites

- [node.js](https://nodejs.org/)
- [Intility NPM Setup](https://github.com/orgs/intility/discussions/13)
- [Visual Studio Code](https://code.visualstudio.com/)
  - When setting up the template, you'll be prompted to install recommended extensions.

## Create App

Run these commands in your terminal.

```bash
npx degit intility/templates/react my-app
cd my-app

npm install
```

Open the project folder in Visual Studio Code with the following command.

```bash
code .
```

You can now start coding by running the start script.

```bash
npm start
```

:::info
The template includes a `.npmrc` config which sets up a custom registry.  
See the [Intility NPM Setup](https://github.com/orgs/intility/discussions/13) guide for how to set up locally.
:::

## Setting up

The template includes some `__X__` variables throughout the codebase.

:::tip Find and Replace
These boxes will appear throughout the setup steps, indicating which variable to replace.
:::

- `__REPO_NAME__`: Name of the GitHub repository
- `__DOCKER_IMAGE_NAME__`: Name of Docker image to push in action
- `__ENTRA_CLIENT_ID__`: Client ID for the app registration
- `__SENTRY_PROJECT_NAME__`: Name of the Sentry project
- `__APP_NAME__`: Kubernetes-friendly name of the app
- `__ARGO_PROJECT__`: Name of the Argo project
